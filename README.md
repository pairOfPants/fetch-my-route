## Fetch My Route
Official repository for Fetch My Route, a Fall 2025 project for CMSC 447 

## Team: Aidan Denham, Alex Marbut, Celestine Sumah, Ethan Michalik

## Tech Stack
- Next.js (React)
- Tailwind CSS (styles)

## Problem Statement
- UMBC only has static maps showing accessible routes for the campus.
- Our goal is to create a navigation system so users can find the fastest accessible route around campus.
